# High-End Visual Design

Engineer agency-level digital experiences.

## Anti-Patterns (Never Do)

- Banned fonts: Inter, Roboto, Arial, Open Sans
- Banned borders: Generic 1px solid gray
- Banned motion: Standard linear/ease-in-out transitions
- Never use h-screen — always min-h-[100dvh]

## Principles

- Double-Bezel nested architecture for all major cards
- Custom cubic-bezier transitions
- Staggered navigation reveals
- GPU-safe animation (transform + opacity only)
