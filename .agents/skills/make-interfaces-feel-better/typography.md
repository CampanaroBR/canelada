# Typography

## Text Wrapping

- text-wrap: balance — headings (≤6 lines)
- text-wrap: pretty — paragraphs, descriptions, captions
- Neither — long text (10+ lines)

## Font Smoothing

html { -webkit-font-smoothing: antialiased; }

## Tabular Numbers

Use font-variant-numeric: tabular-nums for dynamic numbers (counters, prices, scores).
