# Design System — Tokens Primitivos

> Este arquivo define a paleta de cores primitivas do design system.
> Tokens primitivos são os valores brutos — sem semântica de uso ainda.
> Os tokens semânticos (tema claro/escuro via shadcn) serão definidos na próxima seção.
